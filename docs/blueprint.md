# **App Name**: NutriVision AI

## Core Features:

- User Authentication & Profiles: Secure user login, registration, and personalized profile management using Firebase Authentication.
- Food Data Search & Details: Users can search a comprehensive database of food items and view detailed nutritional breakdowns (calories, macros, ingredients, health benefits), with data potentially aggregated or cached in Firestore.
- AI Image Food Recognition: Allow users to upload food images for AI analysis to identify foods and retrieve their nutritional information.
- AI Recipe Generator: Generate tailored recipes based on user-inputted ingredients and dietary preferences using an AI tool.
- AI Nutrition Coaching Chatbot: Provide interactive, personalized nutritional advice and answer food-related questions through an AI-powered chat tool.
- Personalized Dashboard & Saving: Display a personalized user dashboard with saved foods, favorite recipes, and dietary goals, leveraging Firestore for persistence.

## Style Guidelines:

- Light mode primary background color: #ECF5ED. This provides a clean, airy canvas for a healthy living application.
- Primary interactive color: #29993D. A rich, natural green signifying growth, health, and freshness, offering strong contrast against the light background.
- Accent color: #93DE35. A brighter, lively yellowish-green used for highlights and calls to action, drawing the eye without being overpowering.
- Body and headline font: 'Inter' (sans-serif). Note: currently only Google Fonts are supported. Provides a modern, objective, and highly legible aesthetic suitable for data-rich interfaces.
- Card-based UI elements and sections for content display, emphasizing clarity and organization, alongside a responsive design to adapt seamlessly across various screen sizes.
- Use clean, minimalistic line icons that align with the modern and health-focused theme of the application.
- Implement smooth, subtle transitions and micro-animations to enhance user interaction and provide a refined experience, such as loading states or navigation.
